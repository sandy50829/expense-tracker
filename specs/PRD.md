# 產品需求文件 (PRD)

## 1. 產品概述

**產品名稱**: 分帳記帳本 (Expense Tracker)
**版本**: v1.0.0
**目標用戶**: 需要多人分帳記帳的朋友、家人、旅伴

一款類似 Splitwise 的多人記帳分帳 PWA 應用，支援多本帳簿管理、多幣別記錄、即時同步，提供手機友善的繁體中文介面。

## 2. 核心目標

- 讓使用者能快速記錄共同支出並自動計算分帳
- 支援多本帳簿（生活開銷、旅遊、聚餐等）獨立管理
- 多幣別支援，適合跨國旅遊場景
- 即時同步，多人可同時操作
- PWA 封裝，手機上如同原生 APP 體驗

## 3. 目標用戶場景

### 場景 A：日常生活分帳
- 室友共同生活開銷（水電、日用品、伙食）
- 每月結算一次

### 場景 B：旅遊分帳
- 一群朋友出國旅遊
- 不同人在不同時間墊付不同幣別的費用
- 旅程結束後統一結算

### 場景 C：聚餐分帳
- 朋友聚餐，一人先付全額
- 快速記錄後讓大家知道各自分攤金額

## 4. 功能需求

### 4.1 帳號管理
- Email + 密碼註冊/登入
- Google OAuth 第三方登入
- 個人資料設定（顯示名稱、頭像）

### 4.2 筆記本管理
- 建立多本帳簿（名稱、描述、預設幣別、圖示）
- 邀請他人加入帳簿（透過邀請連結/代碼）
- 帳簿成員角色：擁有者 (owner) / 成員 (member)
- 刪除/離開帳簿

### 4.3 記帳功能
- 新增支出：金額、幣別、分類、付款人、日期、備註
- 分帳方式：
  - 均分 (equal)
  - 自訂比例 (percentage)
  - 自訂金額 (exact)
- 支出分類：餐飲、交通、住宿、購物、娛樂、其他
- 編輯/刪除已記錄的支出

### 4.4 多幣別
- 支援主要貨幣：TWD, USD, JPY, EUR, KRW, THB, GBP, CNY, HKD, SGD
- 使用 Frankfurter API 取得每日匯率
- 帳簿內統一以預設幣別顯示換算結果
- 匯率每日快取一次

### 4.5 分帳結算
- 顯示帳簿內每位成員的餘額（應收/應付）
- 最小交易次數演算法，簡化 N 人債務關係
- 顯示「誰該付給誰多少」的清晰列表
- 標記已結算 / 未結算

### 4.6 即時同步
- 多人同時操作時，支出列表即時更新
- 結算狀態即時反映

### 4.7 PWA
- 可安裝至手機主畫面
- 全螢幕顯示（standalone 模式）
- 離線快取靜態資源
- 啟動畫面 (splash screen)

## 5. 非功能需求

### 5.1 效能
- 首次載入 < 3 秒 (LTE 網路)
- 頁面切換 < 300ms
- 支出列表支援虛擬捲軸 (> 100 筆時)

### 5.2 安全性
- Supabase RLS 確保資料隔離
- 使用者只能存取自己所屬帳簿的資料
- 所有 API 呼叫需驗證身份

### 5.3 可用性
- Mobile-first 響應式設計
- 支援 iOS Safari 和 Android Chrome
- 繁體中文介面
- 莫蘭迪色系配色

## 6. 技術架構

- **前端**: Vite + React 19 + TypeScript
- **樣式**: Tailwind CSS v4（莫蘭迪色系自訂主題）
- **路由**: React Router v7
- **狀態管理**: Zustand
- **後端**: Supabase (PostgreSQL + Auth + Realtime)
- **匯率 API**: Frankfurter (api.frankfurter.dev)
- **PWA**: vite-plugin-pwa
- **部署**: Cloudflare Pages
